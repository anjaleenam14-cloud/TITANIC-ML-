Titanic Survival Prediction — Machine Learning Project

Objective-
To predict passenger survival on the Titanic using machine learning models and compare their performance.

Tools Used-

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Google Colab

Steps Performed

1. Data loading and inspection
2. Data cleaning (handled missing values)
3. Feature encoding and selection
4. Model training (Decision Tree, Random Forest, Naive Bayes)
5. Model evaluation and comparison

Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

Best Model

Random Forest Classifier-
(Performs best due to better generalization and reduced overfitting)

Key Insight

* Gender, passenger class, and fare strongly influence survival chances
* Ensemble methods perform better than single models
